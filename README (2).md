## System Requirements

- **Operating System:** Windows 10/11, Linux (Ubuntu 18.04+), macOS 10.15+
- **Python:** 3.8 or higher
- **Memory:** 4GB RAM minimum, 8GB recommended
- **Storage:** 2GB available space
- **Network:** Internet connection for updates and AI features

## Installation

### Quick Install (Recommended)

1. **Windows:**
   ```cmd
   install.bat
   ```

2. **Linux/macOS:**
   ```bash
   chmod +x install.sh
   ./install.sh
   ```

### Manual Install

1. **Run Python installer:**
   ```bash
   python install.py
   ```

2. **Follow the installation prompts**

## Getting Started

1. **Launch the system:**
   ```bash
   cd ~/DiagnosticProSuite
   python CORE/PROFESSIONAL_CONTROL_PANEL.html
   ```

2. **Open your web browser and navigate to:**
   ```
   http://localhost:8080
   ```

3. **Follow the setup wizard to configure your system**

## Configuration

The system configuration is stored in:
- **Windows:** `%USERPROFILE%\DiagnosticProSuite\CONFIG\`
- **Linux/macOS:** `~/DiagnosticProSuite/CONFIG/`

Key configuration files:
- `system_config.json` - Main system configuration
- `ai_config.json` - AI chat settings
- `extensions_config.json` - Extension management
- `tools_config.json` - Support tools configuration

## Troubleshooting

### Common Issues

1. **Installation fails:**
   - Ensure Python 3.8+ is installed
   - Check available disk space (500MB minimum)
   - Run installer as administrator (Windows)

2. **Web interface not accessible:**
   - Check if port 8080 is available
   - Verify firewall settings
   - Restart the system service

3. **AI chat not working:**
   - Check internet connection
   - Verify API keys in configuration
   - Review AI service status

### Log Files

System logs are stored in:
- **Main log:** `LOGS/system.log`
- **AI chat log:** `LOGS/ai_chat.log`
- **Extension log:** `LOGS/extensions.log`
- **Update log:** `LOGS/auto_update.log`
